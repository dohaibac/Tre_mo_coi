== Description ==
Broken Rss Feed Fixer is designed to fix broken rss feeds caused by forcing ads or any other type of extra unwanted elements added into your rss feed,
Broken Rss Feed Fixer will frequently update a new rss feed file called 'rss.xml' you can use this file as your new rss feed file for feedburner or any thing else.


== Installation ==
to install 'Broken Rss Feed Fixer' please follow the instrucions below:

1. Upload `Broken Rss Feed Fixer` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. you will find a new option added into your admin settings menu called 'Rss Feed Fixer'.
4. selsect the 'Rss Update Frequency' period you like 30, 60, 180 or 360 minutes to update your new feed file.
5. 'Broken Rss Feed Fixer' will write new file called 'rss.xml' this will be your new rss feed file you will find this file in the main wordpress dir.
6. note for first time run you have to wait the update frequeny period for the 'rss.xml' file to be created.
