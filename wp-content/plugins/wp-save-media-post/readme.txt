== Description ==

To increase the speed of website. When you save an article, if the images found in post content (The image at another site). This plugin will automatically save the image of the host.

== Installation ==

1. Install 'Save media post' either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Quick link 'Setting' link or goto Settings ->  Option Save Media Post menu. If you want save images to website, You can checked on field 'Save media post' Setting form -> Save Setting
3. That's it. You're ready to go!
